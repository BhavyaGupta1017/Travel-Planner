# Travel-Planner
AI-powered Travel Planner Agent built using IBM watsonx.ai, IBM Granite 4, Flask, HTML, CSS, and JavaScript to generate personalized travel itineraries, hotel recommendations, transport suggestions, and budget estimates.
## 🚀 Features
- 🌍 Personalized travel itinerary
- ✈️ Transport recommendations
- 🏨 Hotel suggestions
- 📅 Day-wise travel planning
- 💰 Budget estimation
- 🍽️ Local food recommendations
- 🧳 Packing checklist
- 🛡️ Safety tips
- 🤖 AI-generated responses using IBM Granite
- 💻 Responsive and user-friendly interface
- 
## 🛠️ Technologies Used
- IBM Cloud Lite
- IBM watsonx.ai Runtime
- IBM Granite 4 H Small Model
- Python
- Flask
- HTML5
- CSS3
- JavaScript
- REST API

- ## 📖 How It Works
1. Enter travel details:
   - Source city
   - Destination
   - Budget
   - Number of days
   - Number of travellers
   - Preferred transport
   - Hotel type
   - Travel interests

2. Click **Generate Travel Plan**.

3. The Flask backend sends the prompt to **IBM Granite** using **IBM watsonx.ai Runtime**.

4. Granite generates a personalized travel itinerary.

5. The itinerary is displayed on the webpage.
